== Description ==

WP e-Commerce Category Shortcut for WP e-Commerce uses information available within the Single Product or All Page WPEC template to display related Products that belong to the same Product Category or Product Tag or optionally a specified Product Category or Product Tag.

WP e-Commerce Category Shortcut extends the WP e-Commerce Plugin by displaying related Products to site. It automatically selects related Products based on the Category or Tag, or these can be specified within the short code.

Thanks to Versipellis, Adam Sargant, code monkey and Onnayo Kheng.

This plugin is based on the WP e-Commerce Related Products plugin by Onnayo Kheng, and provides some additional options and a short code.

== Installation ==

1. Upload the folder 'WP e-Commerce Category Shortcut' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
